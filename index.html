Let's Explore GIT and GitHub

Step to GIT version control system

  • First, create a repository on your GitHub Account
  • Create folder on your local machine
  • Now open the terminal on local machine
  • Navigate to the project folder
  • To initialize git run command -> git init
  • To add all files to git run git add .
  • Now make a commit with commit message git commit -m 'commit_message'
